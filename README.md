# Vue Monorepo with Bun + Tailwind + shadcn (JavaScript)

This repository is a **minimal reproduction / discussion repo** created to ask a question on Reddit about setting up a modern **Vue.js monorepo** using:

* ⚡ **Bun** (package manager + runtime)
* 🧩 **Vue 3**
* 🎨 **Tailwind CSS 4**
* 🧱 **shadcn-vue components**
* 🟨 **JavaScript (no TypeScript)**

The goal is to explore best practices, tooling compatibility, and DX when combining these tools in a monorepo setup.
